# mr-torrent
Discord bot for the bittorrent protocol (downloading linux ISOs)

# Build

`docker build -t mrtorrent .`

# Run

```
export DISCORD_BOT_TOKEN=''
export DISCORD_BOT_PREFIX='!'
export STORAGE_URL=''
export STORAGE_KEY=''
export QBT_URL=''
export QBT_KEY=''
export QBT_PATH=''

# Optional
export DISCORD_ACTIVITY=''
```
